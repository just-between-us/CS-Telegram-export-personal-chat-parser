# **ПАРСЕР ЭКСПОРТА ИЗ ТЕЛЕГРАММА** 
(для гоши)
### 🚀 Quick Start
#### 1. Получить экспорт из Telegram
- Только персональные чаты (Personal chats) в виде json
#### 2. Перенести  
- Полученный файл перенести по пути: `TelegramResultParser\exports`
#### 3. Изменить на свой ID
- Перейти в директорию `Models` 
- Найти `TelegramMessage.cs`
- Изменить значения `MY_USER_ID` и `MY_NAME`
#### 3. Запуск
- Перейти в `cd .\TelegramResultParser\`
- Запустить `dotnet run`
