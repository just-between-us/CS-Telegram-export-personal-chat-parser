# Telegram Personal Chat Export Parser (C#)
(для гоши)
## О проекте

Этот парсер предназначен для обработки файлов, полученных при экспорте личных (персональных) чатов из Telegram Desktop. Он читает JSON-файл, извлекает все сообщения и подготавливает для LLM, отфильтрованными по заданному ID пользователя.

## 🚀 Quick Start

Шаг 1: Получите экспорт из Telegram

1. Откройте Telegram Desktop.
2. Перейдите в настройки: Настройки -> Дополнительно -> Экспорт данных Telegram.
3. В настройках экспорта выберите только личные чаты (Personal Chats) и формат JSON.
4. Запустите экспорт. В результате вы получите папку с файлами, среди которых будет result.json.

Шаг 2: Разместите файл в проекте

1. Склонируйте репозиторий или скачайте исходный код.
2. В папке проекта найдите директорию TelegramResultParser\exports. (Если её нет, создайте вручную).
3. Скопируйте ваш файл result.json (или переименуйте его) в эту папку.

Шаг 3: Укажите свои данные

1. Откройте проект в любом редакторе кода.
2. Перейдите в директорию Models и откройте файл TelegramMessage.cs.
3. Найдите и измените значения двух констант:
   · MY_USER_ID — ваш числовой ID в Telegram.
   · MY_NAME — ваше имя, как оно отображается в чате.

Шаг 4: Запустите парсер

1. Откройте терминал (консоль) в корневой папке проекта (где лежит файл TelegramResultParser.sln).
2. Перейдите в директорию с основным проектом:
```bash 
cd .\TelegramResultParser\
```
3. Выполните команду для запуска:
```bash 
dotnet run
```
## Важные замечания

- Парсер работает только с JSON-экспортом личных чатов. Экспорт каналов или групп не обработаться корректно.
- Убедитесь, что файл называется именно result.json и лежит в папке TelegramResultParser\exports.
- Если вы не знаете свой USER_ID, его можно найти в любом экспорте данных Telegram или с помощью специальных ботов (используйте осторожно).
